# 🩺 Careloom — Doctor Appointment Management System
[![Live Site](https://img.shields.io/badge/Visit-Careloom-blue?style=for-the-badge)](https://careloom-client.vercel.app)

## 📌 Overview

**Careloom** is a modern doctor appointment management platform designed to simplify healthcare accessibility and appointment scheduling. Users can browse doctors from different specialties, view detailed information, securely book appointments, and manage their bookings and profile through an intuitive dashboard.

The platform focuses on creating a smooth and user-friendly healthcare experience with responsive design and secure authentication.

---

## 🎯 Purpose

The goal of **Careloom** is to build a smart and organized healthcare appointment system that allows users to easily discover doctors and schedule appointments online.

The platform aims to:

* Help patients find doctors from multiple specialties
* Provide a seamless appointment booking process
* Deliver secure authentication and user management
* Allow users to manage appointments efficiently
* Create a responsive and engaging healthcare experience

---

## ✨ Key Features

### 🩺 1. Doctor Discovery System

Browse doctors from multiple specialties including:

* Cardiologist
* Neurologist
* Dermatologist
* Pediatrician
* Orthopedic Specialist
* Gynecologist

Users can:

* View doctor information
* Check experience and availability
* View hospital and location details
* Navigate to doctor details pages

---

### 📅 2. Appointment Booking System

Appointment features include:

* Book appointments with preferred doctors
* Select appointment date and time
* Save appointment information to database
* Appointment confirmation through toast notifications

---

### 🔒 3. Authentication & Protected Routes

Secure authentication system with:

* Email & Password Login
* User Registration
* Google Authentication
* JWT Session Management
* Protected Dashboard Routes
* Redirect users to requested pages after login

---

### 👤 4. User Dashboard System

Dashboard functionality includes:

#### My Bookings

* View all booked appointments
* Update appointment details
* Delete appointments
* Instant UI updates

#### My Profile

* View user profile information
* Update profile name
* Update profile photo URL
* Personalized dashboard experience

---

### 🔍 5. Search & User Experience

* Search doctors by name
* Loading spinner while fetching data
* Toast notifications for actions
* Dynamic routing
* Custom 404 page
* Fully responsive design

## 🛠️ Technologies Used

* **Next.js** — Frontend framework & App Router
* **Express.js** — Backend API management
* **MongoDB** — Database management
* **Tailwind CSS** — Utility-first styling
* **DaisyUI** — Responsive UI components
* **JavaScript (ES6+)** — Application logic
* **React Icons** — Icons and UI enhancement
* **React Toastify** — Toast notifications
* **Better Auth** — Authentication management
* **JWT** — Secure authentication/session management
